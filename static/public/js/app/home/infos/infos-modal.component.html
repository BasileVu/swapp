Swap items

Swapp is the website you go to if you have an item you don't need anymore or want to exchange with another one. For example, if you want to swap your Canon DSLR camera against a Nikon camera.
Swapp allows you to simply add items, view items created by others and make offers. It includes social functions such as likes, comments and chat messages.

How does it work ?

Swapp is user friendly and responsive so you shouldn't have too much problem understanding it while using it.
On your first visit on the website you can see and search through the available items. If you are interested in an item or want to propose one of your own item, you must first create an accout and login Then you can create a new item and make offers if you want to.
